# Iwara Downloader

A Firefox extension for downloading videos from iwara.tv quickly and easily.

---

## ✨ Features

- Download videos with a single click or keyboard shortcut (`Alt+Shift+Z`)
- Customize file names and save paths using formats
- Auto-like videos on download
- Update changelog popup on each new version
- Multilingual support

---
