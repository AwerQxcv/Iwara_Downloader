Set download location (can set format)

Sets the path after the download location set in Firefox.

The download location set in Firefox is the value of 'Download location' in the browser's 'about:preferences'.

For example, if the 'Download location' is 'P:\Download' and the set path is 'MMDFans/?username?/', the save location will be 'P:\Download\MMDFans\?username?\filename.mp4'.

Please use '/' to separate folders.

Relative paths like '../' and './' cannot be used.
